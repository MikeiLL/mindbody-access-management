Restrict content of your page using shortcode via access levels based on Mindbody Services.
Includes option for a form which redirects clients to one of three pages: Level 1, Level 2, No Access.

== Description ==

Install and you can limit content based on user MBO memberships:

[mbo-client-access access_levels="1, 2"]
RESTRICTED CONTENT HERE
[/mbo-client-access]

You can also redirect users based on their access level.

== Installation ==

Steps to install and configure MZ MBO Access:

1. Upload the directory, `mz-mbo-access` to the `/wp-content/plugins/` directory
