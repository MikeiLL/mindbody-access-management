Restrict content of your page using shortcode, based on Mindbody memberships

== Description ==

Install and you can limit content based on user MBO memberships:

[mbo-client-access memberships="Corporate Membership"]
RESTRICTED CONTENT HERE
[/mbo-client-access]

== Installation ==

Steps to install and configure MZ Mindbody API:

1. Upload the directory, `mz-mbo-access` to the `/wp-content/plugins/` directory
